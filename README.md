## 📌 About

Backend API for **Leleka** — a pregnancy tracking application.

Provides secure and scalable endpoints for:

- authentication
- user management
- diary
- tasks
- pregnancy data

---

## 🚀 Features

### 🔐 Authentication

- Register
- Login
- Logout
- Token refresh

### 👤 User

- Get current user
- Update profile
- Upload avatar

### ✅ Tasks

- Create tasks
- Get tasks
- Update status

### 📔 Diary

- Create entry
- Get entries
- Update entry
- Delete entry

### 📅 Pregnancy Data

- Current week info
- Baby development
- Mother's condition
- Daily tips

---

## 🛠️ Tech Stack

- Node.js
- Express
- MongoDB + Mongoose
